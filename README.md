# プライバシーポリシーサイト

Googleのプライバシーポリシーページのようなデザインのプライバシーポリシーサイトです。

## 📋 特徴

- **プロフェッショナルなデザイン**: グーグル風の洗練されたUI
- **レスポンシブ対応**: モバイル、タブレット、デスクトップで最適に表示
- **使いやすいナビゲーション**: 目次からのスムーズなスクロール
- **アクセシビリティ**: キーボードナビゲーション対応
- **高速読み込み**: 最小限のCSSとJavaScript

## 📁 ファイル構成

```
policysite/
├── index.html       # メインHTML（プライバシーポリシー内容）
├── styles.css       # スタイルシート
├── script.js        # インタラクティブ機能
└── README.md        # このファイル
```

## 🎨 セクション

1. **概要** - ポリシーの概要と最終更新日
2. **情報の収集** - 収集データの種類
3. **情報の利用** - データ利用の目的
4. **情報の共有** - 第三者への共有について
5. **セキュリティ** - セキュリティ対策
6. **Cookieについて** - Cookie使用について
7. **ユーザーの権利** - ユーザーが持つ権利
8. **お問い合わせ** - 連絡先情報
9. **その他の重要な情報** - 補足事項

## 🚀 使用方法

1. リポジトリをクローン
2. `index.html`をブラウザで開く
3. または、Webサーバーにデプロイ

## ✨ 主な機能

### スムーズスクロール
目次のリンクをクリックすると、スムーズにスクロールします。

### アクティブセクション表示
スクロール時に現在のセクションが目次で自動的にハイライトされます。

### レスポンシブデザイン
全デバイスで最適な表示が可能です。

### キーボードナビゲーション
- `Escキー`で最上部にスクロール

## 🎯 カスタマイズ

### 内容の編集
`index.html`の各セクションを編集してください。

### スタイルの変更
`styles.css`の`:root`セクションで色を変更できます：

```css
:root {
    --primary-color: #1f2937;
    --secondary-color: #4b5563;
    --accent-color: #3b82f6;
    /* ... */
}
```

### 連絡先の更新
`8. お問い合わせ`セクションの連絡先情報を更新してください。
